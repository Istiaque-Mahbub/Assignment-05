What is the Document Object Model (DOM)?

DOM stands for Document Object Model. It gives us the dynamic access of objects.We can go any node or parent node or sibling node or first child or root node using dom.

There are many ways to select an element from dom. 1. document.getELementById()-> where you have to give the id of the desired element 2. document.getElementsByClassName()-> here we have to give the class name 3. document.getElementsByTagName()->this function accepts tag name. 4.document.querySelector()-> we have to use . or # before the id or class getElementsByTagName it gives us the first matching element 5.document.querySelectorAll()->Return all the matching element
